# How to Get Started 🛠️

### Step 1: Clone the Repo and Go to the project directory




```
git clone https://github.com/TulipJani/Zedd.git

cd Zedd

```

### Step 2: Set Up Your .env File

 Google AI API Key: Head over to [Google AI Studio](https://aistudio.google.com/app/apikey) and create an API key. Copy that API and paste it in the .env file
 
 ElevenLabs API Key: Visit [ElevenLabs](https://elevenlabs.io/app/speech-synthesis) and click on bottom left corner for your profile and click on "Profile + API key" and generate an API key if not. Copy that API and paste it in the .env file

Create a .env file in the project root and add your API keys:



```bash
GENAI_API_KEY=your_genai_api_key
ELEVENLABS_API_KEY=your_elevenlabs_api_key
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run Zedd

```bash
python main.py
```
Boom! You're ready to roll. 🎉



# What Zedd Can Do 🤖

## 1. Web and App Interaction

Open Websites: Just name it or drop the URL, and Zedd will open it in a snap.

## 2. Information Retrieval

Weather Updates: Real-time weather info because you gotta know if it's sweater weather.

Current Time: What's the time? Zedd's got you covered.

Google Search: Ask Zedd to Google anything and get the scoop.

Platform-Specific Searches: Amazon, Flipkart, Reddit? You name it, Zedd finds it.

## 3. Media Control

YouTube Streaming: Watch videos or live streams with a simple command.

Spotify Integration: Play your favorite jams directly on Spotify.

## 4. Task Management

To-Do List: Add, mark, and delete tasks from a text-based list. Stay on top of your game!

## 5. Background Music

Calming Music: Play and switch between different chill tracks.

## 6. Calendar Integration

Event Management: Add events with titles, start and end times.

Reminders: Get nudges for your upcoming events.

## 7. News Fetching

News Headlines: Get the latest news across various categories.

## 8. Personalization and Personas

Switch Personalities: Change between different Zedd personalities (e.g., Financial Zedd or Motivational Zedd or Creative Zedd).

## 9. Automation and Control

Enhanced Automation: Automate tasks like music playback, video streaming, and more.

## 10. Command Line Interface

Terminal-Based Operation: A powerful terminal UI for all interactions.
